# Project Name
Pet Care Services Website

## Purpose
This project is a responsive web application for providing pet care services. Users can view different services, read tips, and book appointments online. The goal is to make pet care accessible and informative for all pet owners.


## Key Features
- Responsive design compatible with mobile, tablet, and desktop
- User authentication (Login/Register)
- View detailed service information
- Book services through a simple form
- Loading indicators while data is fetched
- Google sign-in integration

## npm Packages Used
- `react` - For building the user interface
- `react-icons` - Icon components
- `react-toastify` - For showing toast notifications
- `firebase` - Authentication and backend services
- `tailwindcss` - Utility-first CSS framework
- `daisyui` - TailwindCSS component library
